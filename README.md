# MoRestCountriesCA1

Hello!!

This project is a React app that shows information about countries using the Rest Countries API.
You can search countries, view details like population and capital, and filter by region.
It also has dark mode and smooth animations.

I Built with React, Tailwind CSS, DaisyUI, and Framer Motion.
